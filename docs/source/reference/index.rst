==================
Language Reference
==================

This reference describes the ik language precisely. It is the authority on
syntax and semantics; the :doc:`/tutorial/index` is the gentler introduction.

The grammar is given informally throughout and in full, as EBNF, in
:doc:`grammar`. Where this text and the EBNF disagree, the EBNF and the compiler
are correct.
